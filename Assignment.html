Project Title: University Course Management System

Project Overview

The goal of this project is to build a University Course Management System using C++. This system will manage student and course data, handle enrollment, and simulate administrative operations. Students will apply object-oriented principles and utilize key Standard Template Library (STL) containers, including list, pair, map, set, queue, and deque.

Learning Objectives

By completing this project, students will:

  • Implement multiple STL containers in a cohesive software system.

  • Use object-oriented programming to design reusable and modular C++ code.

  • Apply algorithms, iterators, and standard operations on STL containers.

  • Gain experience managing relational data (e.g., students and courses).

Functional Requirements

1. Student Management

  • Add, remove, and update student records.

  • Store student information:

    • Student ID (unique)

    • Full name

    • Enrolled courses (use std::list or std::set)

2. Course Management

  • Add, remove, and update course information:

    • Course code

    • Title

    • Instructor name

    • Enrolled student list (std::list or std::set)

3. Enrollment Operations

  • Enroll and drop students in/from courses.

  • Prevent duplicate enrollments (use std::set).

  • Maintain a waitlist using std::queue if enrollment exceeds course capacity.

4. Administrative Tools

  • List all courses a student is enrolled in.

  • List all students in a specific course.

  • Log enrollments and drops using std::deque for fast front/back operations.

  • Support search operations with std::map and std::pair.

Technical Specifications

  • Language: C++ (Standard C++17 or higher)

  • STL containers to use:

    • std::map for student and course registries.

    • std::list or std::set for enrollment lists.

    • std::pair for key-value pairs and associations.

    • std::queue for course waitlists.

    • std::deque for activity logs.
